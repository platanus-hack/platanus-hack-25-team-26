¿Qué es Ingeniería Social?

Son técnicas de manipulación psicológica que usan atacantes para engañar a las personas y obtener información, dinero o acceso a sistemas.

Recomendaciones para proteger a tu familia

  • Conversa con tu hijo/a sobre mensajes que crean urgencia o miedo artificial.
  • Enséñale a desconfiar de ofertas "demasiado buenas para ser verdad" o premios inesperados.
  • Verifica siempre la identidad de quien solicita información personal o dinero.
  • Ninguna empresa legítima pide contraseñas, códigos de verificación o datos bancarios por mensaje.
  • Si algo genera presión para decidir "ahora mismo", es una señal de alerta.
